# Micro Panda

Micro service made easy!

## Status

Under active development.

## Goals

- transparent transport
- easy to use
  - high level abstraction
  - modern api
  - zero config
- extensible
  - plugin - provide abilities
  - middleware - provide controls
- eco-system
  - transporter
  - api gateway
  - logger
  - error handling
  - monitor
- message model
  - one to one req-res
  - one to many via topic
  - one to group via topic
